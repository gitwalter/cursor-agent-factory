### AI-Powered Software Dev

Comprehensive resource covering:
- **Multi-Agent System Design**: Architecture patterns for orchestrating multiple specialized AI agents
- **MCP Server Integration**: Model Context Protocol servers and their configuration
- **Best Practices**: Verified methodologies for AI-assisted development including prompt engineering, session management, and output verification
- **Tool Catalogs**: Available tools and frameworks for building agentic systems

**Recommended For**: Workshop facilitators, developers seeking deep dives into multi-agent architecture.

### AI Agent Dev Start

Foundational education covering:
- **Agentic Spectrum**: Understanding the difference between Workflows (structured automation) and Agents (autonomous problem-solvers)
- **LangChain/LangGraph**: Building and orchestrating LLM systems
- **Workflow Patterns**: Prompt chaining, routing, and other structured automation patterns
- **Developer-Focused**: Practical, hands-on orientation with real examples

**Recommended For**: Pre-reading for workshops, Express Onboarding deep-dive reference.
