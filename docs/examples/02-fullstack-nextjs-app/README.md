# Example 02: Full-Stack Next.js App

Build a modern task management application with Next.js 14+, React Server Components, Prisma ORM, and authentication.

## What We're Building

A **Task Management Application** that provides:

- User authentication (NextAuth.js)
- Task CRUD operations with real-time updates
- Project organization and categorization
- Responsive UI with Tailwind CSS
- Server-side rendering for performance

## Target Users

- **Primary**: Full-stack developers building React applications
- **Secondary**: Frontend developers learning Next.js patterns

## Success Criteria

- Full-stack feature implementation in < 2 hours
- Type-safe from database to UI
- 90% Lighthouse performance score

## Technology Stack

| Component | Technology |
|-----------|------------|
| Framework | Next.js 14+ (App Router) |
| Language | TypeScript 5+ |
| Database | PostgreSQL with Prisma |
| Auth | NextAuth.js |
| Styling | Tailwind CSS |
| Testing | Vitest, Playwright |
